System Overview

Gym Attendance Management System

A simple, reliable, and fully offline system for managing your gym members and tracking daily attendance — built specifically for your business.

What's Included

Core Features

Everything your gym needs to track members and attendance, with no internet required.

🖥️

Self-Service Kiosk

Members type their name or ID to check in and check out. Simple touchscreen-friendly interface.

👥

Member Management

Add, edit, and deactivate members. Store their name, contact number, and membership details.

📋

Attendance Records

Every check-in and check-out is automatically logged with a timestamp.

🔍

Search & Filter

Search attendance by member name or filter by date range to find records quickly.

📊

CSV Export

Export attendance data to a spreadsheet file anytime for reporting or record-keeping.

🔐

Admin Login

Password-protected admin panel. Only authorized staff can manage members and view reports.

Daily Use

How It Works

Designed to be straightforward for both members and staff.

1

PC is turned on — system starts automatically

The kiosk screen launches on startup. No manual setup needed each day.

2

Member checks in at the kiosk

They type their name or member ID and tap "Check In." The time is recorded instantly.

3

Member checks out when leaving

Same process — the system logs their check-out time automatically.

4

Admin reviews records anytime

Staff logs into the admin panel to view attendance, manage members, or export reports.

Technical Details

System Specifications

Built to run reliably on standard hardware with no ongoing costs.

Platform Windows PC (desktop or laptop)
Internet Required No — fully offline, runs on your local machine
Database Local file-based database (SQLite) — stored on your PC
Access Via any web browser on the same PC (Chrome, Edge, Firefox)
Startup Auto-starts when the PC is turned on
Backup Auto-backup on every startup (keeps last 7). Optional: auto-copy to USB flash drive and/or Google Drive for Desktop.
Monthly Cost None (no subscriptions, no cloud fees)
Important Notes

System Limitations

Please read these before signing the service agreement.

  • Single location only. This system is designed for one gym on one PC. It does not support multiple branches sharing the same data.
  • No remote access. The system cannot be accessed from a phone, tablet, or another PC outside the gym network.
  • Optional cloud backup only. Google Drive backup is available if Google Drive for Desktop is installed — but this requires the client to maintain their own Google account. The system does not guarantee cloud availability.
  • No mobile app. There is no companion app for members or admins on smartphones.
  • Expansion requires a new project. Adding branches, cloud hosting, or remote access is outside the scope of this system and will be a separate engagement.
