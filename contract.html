1. Scope of Work

The Developer agrees to design, build, and deliver a local gym attendance management system ("the System") with the following features:

  • Member registration and management (add, edit, deactivate)
  • Kiosk-based check-in and check-out
  • Admin dashboard with attendance records
  • Search, filter, and CSV export of attendance data
  • Payment recording and payment history per member
  • Secure admin login
  • Auto-start script for Windows PC

2. System Limitations

The Client acknowledges that the System is built for:

  • Single location use only. The System operates on one (1) PC and does not support multiple branches or remote access.
  • Local network only. No internet connection or cloud storage is included.
  • Single gym deployment. This license covers one (1) installation only.

Expansion to multiple branches, cloud hosting, or additional installations will be treated as a separate project and billed accordingly.

3. Deliverables
